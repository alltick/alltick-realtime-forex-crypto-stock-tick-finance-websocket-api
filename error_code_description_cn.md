# 错误码说明

| 错误码 | 含义                         |
| ------ | ---------------------------- |
| 200    | ok                           |
| 400    | request header param invalid |
| 400    | request data param invalid   |
| 401    | token invalid                |
| 429    | rate limit                   |
| 600    | code invalid                 |
| 601    | body empty                   |
