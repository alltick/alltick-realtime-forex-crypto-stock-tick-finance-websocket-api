# 简介
本文档主要目的是对外提供免费的行情接口快速接入方法。
主要功能如下：
- 获取港股实时行情API
- 获取港股实时10档盘口API
- 获取港股K线数据API
- 获取美股实时行情API
- 获取美股实时一档盘口API
- 获取美股K线数据API
- 获取沪深A股实时行情API
- 获取沪深A股实时5档盘口API
- 获取沪深A股K线数据API
- 获取加密货币实时行情API
- 获取加密货币实时多档盘口API
- 获取加密货币K线数据API
- 获取外汇实时行情API
- 获取外汇实时5档盘口API

- 获取外汇K线数据API

- 获取贵金属实时行情API

- 获取贵金属实时5档盘口API

- 获取贵金属K线数据API

# 接口分类
接口分为http接口和websocket接口两大类。

## http接口主要提供：
- K线查询
- 批量K线查询
- 最新盘口报价查询
- 最新成交报价查询
  
## websocket接口主要提供：
- 心跳
- 盘口报价订阅
- 成交报价订阅
- 订阅取消
